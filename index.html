Estilos en CSS
  1. Introducción

    2. ¿Qué es CSS?
    CSS(Cascade Style Sheets)(Hojas de estilo en cascada) nos sirve para crear diversos estilos para nuestros eleentos en HTML. Pudiendo definir de forma estructurada la manera en la cual se organizarán estos estilos.
    CSS nos permite ahorrar mucho trabajo, ya que al crear una plantilla de estilos que puede ser aplicada rápidamente, evitamos tener que crear un estilo individual para cada elemento.

  2. ¿Cómo usarlo?

    3. Existen tres formas de utilizar CSS:

    • Externa
      • Para utilizar el código CSS de forma externa podemos utilizar un archivo diferente dedicado exclusviamente al código CSS.
      Para ello utilizamos la etiqueta:
      <link rel="stylesheet" href="miestilo.css">
    • Interna
      • También podemos usar el código CSS en nuestra página HTML usando la etiqueta:
      <style>
      body(color: green);
      </style>
