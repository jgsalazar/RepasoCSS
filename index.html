ETIQUETAS DE ESTRUCTURACIÓN

<hi> CABECERA

Volver

Se utilizan para definir la cabecera o título de una página o sección, y cuenta con 6 niveles de importancia o tamaños: h1, h2, h3, h4, h5 y h6. Nótese que se añade de forma automática un espacio antes y después de cada cabecera. Ejemplos:

Código:

<hi>Cabecera hi</hi>,
donde: i = 1, 2, 3, 4, 5 o 6.

Resultado:

Cabecera h1

Cabecera h2

Cabecera h3

Cabecera h4

Cabecera h5
Cabecera h6

<p> PÁRRAFO

Volver

Se utilizan para definir los párrafos en nuestro documento. Nótese que se añade de forma automática un espacio antes y después de cada párrafo. Ejemplo:

Código:

<p>Párrafo 1: Lorem ipsum dolor sit amet consectetur adipisicing elit. Doloribus libero sed placeat autem debitis doloremque, similique aspernatur. Quasi, eveniet ullam.</p>
<p>Párrafo 2: Lorem ipsum dolor sit, amet consectetur adipisicing elit. Nemo, molestiae cupiditate! Consequuntur ad magni assumenda amet impedit dolorem, dolor provident quo eos! Corrupti, quis mollitia!</p>

Resultado:

Párrafo 1: Lorem ipsum dolor sit amet consectetur adipisicing elit. Doloribus libero sed placeat autem debitis doloremque, similique aspernatur. Quasi, eveniet ullam.

Párrafo 2: Lorem ipsum dolor sit, amet consectetur adipisicing elit. Nemo, molestiae cupiditate! Consequuntur ad magni assumenda amet impedit dolorem, dolor provident quo eos! Corrupti, quis mollitia!

<br> SALTO DE LÍNEA

Volver

Se utiliza para comenzar una nueva línea de texto dentro un mismo párrafo. Nótese que en este caso no se necesita la etiqueta de cierre. Ejemplo:

Código:

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Perspiciatis, facilis!<br> Lorem ipsum dolor sit amet consectetur adipisicing elit.<br> Lorem ipsum dolor sit amet consectetur.

Resultado:

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Perspiciatis, facilis!
Lorem ipsum dolor sit amet consectetur adipisicing elit.
Lorem ipsum dolor sit amet consectetur.

<section> SECCIÓN

Volver

Se utiliza para definir un bloque dentro el contenido, al cual se le podrá dar características diferentes al resto del texto.

<div> DIVISIÓN

Volver

Se utiliza para definir una división dentro el contenido, al cual se le podrá dar características diferentes al resto del texto. Es similar a la etiqueta <section>

ETIQUETAS DE FORMATO DE TEXTO

<code> CÓDIGO

Volver

Esta etiqueta otorga al texto un estilo de fuente de ancho fijo, y se utiliza para mostrar funciones, código de programación o variables. Ejemplo:

Código:

El siguiente texto está marcado como código: <code>System.out.println(¡Hola Mundo!)</code>

Resultado:

El siguiente texto está marcado como código: System.out.println(¡Hola Mundo!)

<b> NEGRITA

Volver

Código:

El siguiente texto está en negrita: <b>Texto en negrita</b>

Resultado:

El siguiente texto está en negrita: Texto en negrita

<sub> SUBÍNDICE

Volver

Esta etiqueta permite insertar texto en subíndice, es especialmente útil en fórmulas químicas. Ejemplo:

Código:

En H<sub>2</sub>O el número 2 está en subíndice

Resultado:

En H2O el número 2 está en subíndice

<sup> SUPERÍNDICE

Volver

Esta etiqueta permite insertar texto en superíndice, es especialmente útil en fórmulas matemáticas. Ejemplo:

Código:

Los exponentes del siguiente polinomio están en superíndice, x<sup>4</sup>+5x<sup>3</sup>-3x<sup>2</sup>

Resultado:

Los exponentes del siguiente polinomio están en superíndice, x4+5x3-3x2

ETIQUETAS DE LISTAS

<ul> LISTAS NO ORDENADAS

Volver

Las listas no ordenadas se utilizan para nombrar elementos sin un orden especifico.

Los atributos para definir esta lista son los siguientes:

<ul>
Para definir la lista
<li>
Para añadir un nuevo elemento

Ejemplo:

Código:

<ul>Departamentos de Bolivia
<li>La Paz</li>
<li>Cochabamba</li>
<li>Santa Cruz</li>
<li>Oruro</li>
<li>Potosí</li>
<li>Chuquisaca</li>
<li>Tarija</li>
<li>Beni</li>
<li>Pando</li>
</ul>

Resultado:

    Departamentos de Bolivia:
  • La Paz
  • Cochabamba
  • Santa Cruz
  • Oruro
  • Potosí
  • Chuquisaca
  • Tarija
  • Beni
  • Pando

<ol> LISTAS ORDENADAS

Volver

Las listas ordenadas se utilizan para nombrar elementos con un orden especifico.

Los atributos para definir esta lista son los siguientes:

<ol>
Para definir la lista
<li>
Para añadir un nuevo elemento

Ejemplo:

Código:

<ol>Pasos para lavarse las manos:
<li>Mojarse las manos.</li>
<li>Cerrar el grifo y enjabonarse las manos.</li>
<li>Frotarse las manos hasta hacer espuma.</li>
<li>Restregarse las manos durante al menos 20 segundos.</li>
<li>Enjuagarse bien las manos.</li>
</ol>

Resultado:

    Pasos para lavarse las manos:
  1. Mojarse las manos.
  2. Cerrar el grifo y enjabonarse las manos.
  3. Frotarse las manos hasta hacer espuma.
  4. Restregarse las manos durante al menos 20 segundos.
  5. Enjuagarse bien las manos.

<dl> LISTAS DE DEFINICIONES

Volver

Las listas de definiciones se utilizan cuando queremos mostrar un glosario de términos o conceptos.

Los atributos para definir esta lista son los siguientes:

<dl>
Para definir la lista
<dt>
Para añadir un nuevo término
<dd>
Para añadir la definición del término

Ejemplo:

Código:

<dl>Comandos en Word:
<dt>Ctrl+O</dt>
<dd>Abre un documentos.</dd>
<dt>CtrlC</dt>
<dd>Copia el texto seleccionado.</dd>
<dt>Ctrl+X</dt>
<dd>Corta el texto seleccionado.</dd>
<dt>Ctrl+V</dt>
<dd>Pega el texto previamente copiado o cortado.</dd>
</dl>

Resultado:

Comandos en Word:
Ctrl+O
Abre un documento.
Ctrl+C
Copia el texto seleccionado.
Ctrl+X
Corta el texto seleccionado.
Ctrl+V
Pega el texto previamente copiado o cortado.

INSERCIÓN DE IMÁGENES Y VIDEOS

<img> IMÁGENES

Volver

Esta etiqueta permite insertar imágenes.

Los atributos que se utilizan son los siguientes:

<img>
Para indicar la inserción de la imagen.
src
Para definir la ubicación de la imagen a insertar.
width
Para definir el ancho de la imagen en pixeles (px).
alt
Para mostrar un texto en caso de que la imagen no se muestre por algún motivo o error.

Ejemplo:

Código:

<img src="img/logo.png" width="100px" alt"Error en la imagen">
<img src="img/logo.png" width="50px" alt"Error en la imagen">

Resultado:

<video> VIDEO PROPIO

Volver

Esta etiqueta permite insertar un video guardado en nuestra computadora.

Los atributos que se utilizan son los siguientes:

<video>
Para indicar la inserción del video.
width
Para definir el ancho del video.
height
Para definir el alto del video.
poster
Para definir la ubicación de una imagen que se muestre mientras el video no sea reproducido.
controls
Añade los botones de control de reproducción del video.
<source>
Para definir el origen del video.
src
Para definir la ubicación del video a insertar.
type
Para definir el formato del video

Ejemplo:

Código:

<video width="720" height="480" poster="img/video propio.png" controls> <source src="img/Video1.mp4" type="video/mp4"> Ha ocurrido un error al mostrar el video, su navegador no es compatible. </video>

Resultado:

<video> VIDEO EMBEBIDO

Volver

Esta etiqueta permite insertar un video alojado en una página web externa, evitando tener que descargarlo.

Los atributos que se utilizan son los siguientes:

<iframe>
Para indicar la inserción de un video embebido.
width
Para definir el ancho del video.
height
Para definir el alto del video.
src
Para definir la ubicación web del video a insertar.
title
Para definir el título del video.
frameborder
Para dar un borde al video.
allow
Para otorgar permisos adicionales.
allowfullscreen
Para permitir la reproducción en pantalla completa.

Ejemplo:

Código:

<iframe width="480" height="320" src="https://www.youtube.com/embed/QHAB3UvgBI8" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen> </iframe>
